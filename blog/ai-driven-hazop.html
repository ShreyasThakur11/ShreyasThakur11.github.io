Automating Process Hazard Analysis with Rule-Based AI

Published on Feb 21, 2026 â€¢ 6 min read

Process Hazard Analysis (PHA), specifically the HAZOP study, is a critical step in chemical plant safety. Conventionally, this involves a multidisciplinary team sitting in a room for weeks, brainstorming "What If" scenarios. The AI HAZOP Assistant I built aims to accelerate this process by automating the preliminary generation of deviations and causes.

The Logic: Expert Systems vs. LLMs

While Large Language Models (LLMs) like GPT-4 are powerful, they can hallucinate safety-critical facts. For industrial safety, determinism is key. Therefore, I chose a **Rule-Based Expert System** architecture.

The system relies on a structured Knowledge Base (KB) derived from CCPS guidelines and industrial experience. It maps:

  • Equipment (e.g., Reactor) → Parameters (Temp, Pressure)
  • Parameter (Flow) + Guideword (No) → Deviation (No Flow)
  • Deviation → Causes (Pump Trip) → Consequences (Runaway)

NLP for Node Extraction

To make the tool user-friendly, I implemented a lightweight Natural Language Processing (NLP) parser. It scans the user's process description for keywords to identify equipment nodes using simple heuristic matching.

// Node Extraction Logic const NODES = { "reactor": { params: ["Temp", "Pressure"] }, "pump": { params: ["Flow", "Pressure"] } }; function extractNodes(text) { const found = []; Object.keys(NODES).forEach(key => { if(text.toLowerCase().includes(key)) { found.push({ type: key, ...NODES[key] }); } }); return found; // Returns list of equipment to study }

Risk Ranking Matrix

The tool also estimates risk using a simplified Severity x Likelihood matrix. For example, if the node is a "Reactor" and the deviation is "High Temperature", the system logic flags this as High Severity due to the potential for a runaway reaction, automatically assigning a "High Risk" badge.

Conclusion

This tool demonstrates how we can capture domain knowledge into software to assist engineers. It doesn't replace the human HAZOP team, but it provides a robust "first draft" to ensure no obvious hazards are missed.
